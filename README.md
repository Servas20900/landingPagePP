# Landing Page PP

Proyecto MVP de landing comercial en Next.js + Tailwind CSS para PP Promotora.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Rutas

- `/` Landing one-page
- `/eventos/[slug]` Detalle dinámico de evento

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
npm run start
```
